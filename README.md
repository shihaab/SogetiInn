# To use SogetiInn API
Navigate to target folder in CMD.

## Clone repository
```
gh repo clone shihaab/SogetiInn
```

## Install dependencies
```
pip install Flask
```
```
pip install Faker
```
```
pip install db-sqlite3
```
## Run application
```
flask --app main run
```
## Application should be running on 

http://127.0.0.1:5000/
