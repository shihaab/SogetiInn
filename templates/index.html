SogetiInn API Documentation

Building an API call

http://127.0.0.1:5000/ bookings ? room_id = 1 & client_id = 9

/bookings

Get a list of bookings based on various parameters.

  • room_id: Room ID (optional)
  • client_id: Client ID (optional)
  • coupon_used: Coupon Used (optional)
  • start_date_time: Start Date and Time (optional)
  • end_date_time: End Date and Time (optional)

/cleaning_employees

Get a list of cleaning employees based on various parameters.

  • employee_number: Employee Number (optional)
  • first_name: First Name (optional)
  • last_name: Last Name (optional)
  • dob: Date of Birth (optional)
  • phone_number: Phone Number (optional)

/clients

Get a list of clients based on various parameters.

  • first_name: First Name (optional)
  • last_name: Last Name (optional)
  • dob: Date of Birth (optional)
  • phone_number: Phone Number (optional)
  • iban: IBAN (optional)

/complaints

Get a list of complaints based on various parameters.

  • booking_id: Booking ID (optional)
  • description: Complaint Description (optional)

/receipts

Get a list of receipts based on various parameters.

  • tab_id: Tab ID (optional)
  • amount: Receipt Amount (optional)
  • currency: Currency (optional)

/rooms

Get a list of rooms based on various parameters.

  • number_on_floor: Number on Floor (optional)
  • floor: Floor (optional)
  • occupied: Occupied (optional)
  • last_cleaned: Last Cleaned (optional)
  • last_cleaned_employee_id: Last Cleaned Employee ID (optional)
  • amenities: Amenities (optional)

/tabs

Get a list of tabs based on various parameters.

  • booking_id: Booking ID (optional)
  • status: Tab Status (optional)

/open-tabs

Get a list of open tabs for a specific booking.

  • booking_id: Booking ID (required)

/available-rooms

Check the availability of rooms based on the booking information.

  • date: Date in YYYY-MM-DD format (required)

/reset-db

Reset and populate the database.

/checkout-service

Get the total amount of the booking.

  • booking_id: Booking ID (required)

/create-booking

Create a new booking.

  • room_id: Room ID (required)
  • client_id: Client ID (required)
  • start_date_time: Start Date and Time in YYYY-MM-DD format (required)
  • end_date_time: End Date and Time in YYYY-MM-DD format (required)
  • coupons_used: Coupons Used (optional)

/create-client

Create a new client.

  • first_name: First Name (required)
  • last_name: Last Name (required)
  • dob: Date of Birth in YYYY-MM-DD format (required)
  • phone_number: Phone Number (required)
  • iban: IBAN (required)

/create-tab

Create a new tab.

  • booking_id: Booking ID (required)
  • status: Tab Status (required)

/create-receipt

Create a new receipt.

  • tab_id: Tab ID (required)
  • amount: Receipt Amount (required)
  • currency: Currency (required)

/remove-booking

Remove a booking.

  • id: Booking ID (required)

/remove-client

Remove a client.

  • id: Client ID (required)

/remove-receipt

Remove a receipt.

  • id: Receipt ID (required)

/update-booking

Update a booking.

  • room_id: ID (required)
  • room_id: Room ID (optional)
  • client_id: Client ID (optional)
  • coupon_used: Coupon Used (optional)
  • start_date_time: Start Date and Time (optional)
  • end_date_time: End Date and Time (optional)

/update-receipt

Update a receipts.

  • tab_id: Tab ID (optional)
  • amount: Receipt Amount (optional)
  • currency: Currency (optional)

/update-room

Update a room.

  • number_on_floor: Number on Floor (optional)
  • floor: Floor (optional)
  • occupied: Occupied (optional)
  • last_cleaned: Last Cleaned (optional)
  • last_cleaned_employee_id: Last Cleaned Employee ID (optional)
  • amenities: Amenities (optional)
